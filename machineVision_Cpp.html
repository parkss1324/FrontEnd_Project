Machine-Vision

"머신비전(machine-vision)"은 기계에 인간의 시각과 판단 기능을 부여하여,
사람이 인지하고 판단하는 작업을 하드웨어와 소프트웨어 시스템이 대신 수행하는 기술입니다.

이는 카메라, 렌즈, 조명, 이미지 프로세서, 소프트웨어 등으로 구성된 시스템을 통해 이루어지며,
제조업, 물류, 의료, 자동차 등 다양한 분야에 적용됩니다.

머신비전을 통해 제품 품질을 개선하고 안정성을 높이며, 생산성 및 정밀도를 향상시키는 역할을 합니다.

2. yt-dlp 라이브러리 설치

라이브러리

프로그래밍에서 자주 사용되는 코드(함수, 클래스, 모듈 등)를 모아 놓은 재사용 가능한 코드 집합입니다.
특정 작업을 쉽게 수행할 수 있도록 미리 작성된 코드 모음으로, 개발자가 처음부터 코드를 작성하지 않아도 되도록 도와줍니다.

yt-dlp

Youtube 및 기타 여러 비디오 플랫폼에서 동영상을 다운로드하고 다양한 정보를 추출할 수 있는 강력한 오픈 소스 Python 라이브러리 및 명령줄 도구입니다.
이는 원래 youtube-dl 프로젝트를 기반으로 개발되었으며, 더 많은 기능과 개선사항을 추가한 포크(Fork) 버전입니다.

5. 느낀 점(회고)

새로운 개발 환경에 대한 도전 OpenCV를 동작시키는 Visual Studio가 MacOS 지원을 중단하여 개발이 가장 마지막으로 이루어졌습니다.

Visual Studio Code를 통해 동작시키기 위해 시간이 날 때마다 관련 자료들을 찾아보기 시작했고 결국 동작시킬 수 있었습니다.

현재는 이를 활용하여 다양한 머신비전 기능 개발을 진행하고 있습니다.
사용 분야의 범위 응용 영상에 대해 기능 구현을 할 때, URL 영상 주소만 바꿔서 원하는 영상으로 테스트할 수 있습니다.

차선 검출 외에도 코드를 얼굴 또는 번호판 인식으로 코드를 바꿔서 응용할 수 있습니다.
추가 기능 구현 계획 개발 영역 확장을 위해 머신러닝, 딥러닝 기술을 적용시킬 예정입니다.

여러 도로 주행 영상을 학습시켜, 조금 더 정확한 검출이 가능하도록 하기 위해 머신 비전 알고리즘을 공부하고 있습니다.

영상 관련 알고리즘을 적용시켜 차선 검출 외에도 다양한 머신러닝, 딥러닝 개발을 계획하고 있습니다.
