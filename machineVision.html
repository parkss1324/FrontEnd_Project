Machine-Vision

"머신비전(machine-vision)"은 기계에 인간의 시각과 판단 기능을 부여하여,
사람이 인지하고 판단하는 작업을 하드웨어와 소프트웨어 시스템이 대신 수행하는 기술입니다.

이는 카메라, 렌즈, 조명, 이미지 프로세서, 소프트웨어 등으로 구성된 시스템을 통해 이루어지며,
제조업, 물류, 의료, 자동차 등 다양한 분야에 적용됩니다.

머신비전을 통해 제품 품질을 개선하고 안정성을 높이며, 생산성 및 정밀도를 향상시키는 역할을 합니다.

라이브러리

프로그래밍에서 자주 사용되는 코드(함수, 클래스, 모듈 등)를 모아 놓은 재사용 가능한 코드 집합입니다.
특정 작업을 쉽게 수행할 수 있도록 미리 작성된 코드 모음으로, 개발자가 처음부터 코드를 작성하지 않아도 되도록 도와줍니다.

mediapipe

Google에서 개발한 강력한 머신 러닝 라이브러리로, 컴퓨터 비전과 동작 추적 관련 작업을 쉽게 구현할 수 있도록 설계 되었습니다.
주로 사람의 움직임, 얼굴, 손, 포즈, 객체 등을 실시간으로 감지하거나 추적하는 데 사용됨니다.

numpy

파이썬에서 과학 연산과 데이터 처리를 위한 라이브러리입니다.
주로 다차원 배열을 사용하여 수치 데이터를 효율적으로 처리하며, 선형대수, 통계 계산, 데이터 분석 등의 작업에서 많이 사용됩니다.

다른 개발 영역의 도전 임베디드 개발자로 지내며, 다른 개발 분야에 대한 호기심은 있었지만, 어디서부터 시작해야 될지 막막해 미루어 왔습니다.

퇴사 후, 취업 준비를 하는 과정에서 다양한 개발 분야의 강의에 대해 알게 되었고 프론트엔드에 대해 본격적으로 배울 수 있게 되었습니다.

처음 시작은 힘들지만, 새로운 도전은 이루었을 때 큰 성취감을 가져다 주었습니다.
개발 지식 확장 및 소프트웨어 활용 능력 향상 HTML, CSS, JavaScript 등 프론트엔드를 대표하는 개념에 대해 알게 되고,
코드를 더 자세히 보기 위해서만 활용했던 'Visual Studio Code'를 'GitHub'와 연동시켜 본격적인 개발 활동을 시작하게 되었습니다.

GitHub가 제공하는 웹 호스팅 기능과 VS Code에서 제공하는 'Live Server'를 활용하여
작성한 내용이 웹 페이지에서 변경되는 것을 확인했을 때, 더 전문적인 개발자가 될 수 있었습니다.
새로운 개발의 재미 부품과 소프트웨어마다 데이터시트와 명령어가 달라 개발환경설정에 많은 어려움을 겪었던 임베디드와 달리,
프론트엔드는 큰 제약없이 노트북만으로 원하는 결과를 확인할 수 있어서 배우는 것이 되게 즐거운 시간이었습니다.
